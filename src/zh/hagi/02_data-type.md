# B. 值和类型

值有很多种类型。例如42和“Zeile”不是一个类型。Zeile中最基础的类型有4种，剩下的类型会更高级。

| 类型 | 代表 | 例子 | 备注 |
| --- | --- | --- | --- |
| Int | 整数 | 42, -114, 0 | |
| Float | 浮点数 | 3.2, 9, -1.4,| 小数。Float也可以存整数，但是没必要。|
| String | 字符串 | "Farbe", "色彩", "いろ" | 文本。必须写在`""`（两个直着的引号）内。 |
| Bool | 布尔值 | true, false | 真或假。只有这两种可能。 |

## 字符串

字符串需要写在`""`之间，例如`"你好"`。引号必须是直引号（笔画垂直向下，而非倾斜：`“”`），否则计算机不能理解。

`""`必须在一行内结束。你可以用`"""`来表达多行字符串，例如：

```swift
let songName = "Sanctuary"

let lyrics = """
Dearest
Your are my dearest
"""
```

`songName`只有一行，而`lyrics`有两行。

反斜杠（`\`）用于表示特殊内容。这样子表达特定字符被称作“转译”。

字符串内可以用`\n`换行。例如`Dearest\nYour are my dearest`和上述代码的`lyrics`相等。

| 转译字符 | 含义 |
| --- | --- |
| `\n` | 换行 |
| `\"` | 双引号 |
| `\'` | 单引号 |
| `\\` | 反斜杠 |
| `\0` | 空字符 |

```swift
"\\あなた達、Roselia\0にすべてを賭ける覚悟はある？\n\"はい！\""
```

相当于

> \あなた達、Roseliaにすべてを賭ける覚悟はある？
> "はい！"
